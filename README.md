Theano-Kaldi
============

A solution for training and decoding neural networks with unique structure
using the Kaldi toolkit.

## Usage
```
cd <dataset>/s5
git clone --recursive git@github.com:shawntan/theano-kaldi.git
theano-kaldi/run.sh
```
